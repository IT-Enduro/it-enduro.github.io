## Общая информация

Воркшоп по Docker — эффективное сочетание структурированной теории и практики. На занятии участники работают со своего
ноутбука и в реальном времени разбирают нюансы Docker вместе с преподавателем.

Трехчасовое практическое занятие для разработчиков, QA инженеров, системных администраторов и DevOps.

## Описание курса

Воркшоп помогает разобраться с Docker и увереннее чувствовать себя в рабочих задачах и на технических собеседованиях.

- **Удобный формат**: можно подключиться онлайн из любой точки мира.
- **3 часа**: структурированная теория и практические навыки.
- **Практика**: ответы на вопросы и разбор проблем по ходу занятия.

## Программа

- Введение и предпосылки контейнеризации.
- Архитектура Docker.
- Внутреннее устройство Docker: cgroups и namespaces.
- Основные команды docker cli.
- Volumes.
- Dockerfile.
- Dockerfile для статического сайта.
- Сетевое взаимодействие контейнеров.
- Docker Compose и запуск нескольких контейнеров.
- Базовая оркестрация в Docker Swarm.

## По итогам интенсива вы

- Поймете, как устроен Docker внутри.
- Освоите работу с Docker CLI и сборку образов.
- Научитесь запускать мультиконтейнерные сценарии в Docker Compose и Docker Swarm.
